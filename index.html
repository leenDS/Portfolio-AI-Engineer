PROJECT_01
Multi-Source Autonomous RAG Platform
Production Agentic
Situation
Enterprise clients required intelligent Q&A over heterogeneous document corpora — PDFs, CSVs, HTML tables, and structured databases — with no unified retrieval layer.
Task
Architect a multi-source RAG pipeline capable of autonomous document routing, semantic reranking, and persona-adaptive response generation at sub-200ms latency.
Action
Designed an ingestion pipeline using Unstructured.io + Haystack 2.0 for document parsing. Implemented hybrid retrieval (BM25 + dense vectors) in Qdrant with BGE-Reranker cross-encoder scoring. Built a dynamic AI Persona layer that tailors tone, depth, and citation style per user role.
Result
Achieved 94% retrieval precision@5 on internal benchmarks. Persona generation reduced user follow-up queries by 38%. Platform now processes 50K+ documents across 6 enterprise clients.

PROJECT_02
Sign2Sign — YOLO to 3D Avatar Evolution
🥇 1st Place P2M R&D
Situation
The deaf and hard-of-hearing community in the Arab world lacked real-time, bidirectional sign language translation tools — existing solutions were static, non-interactive, and not culturally adapted to Arabic Sign Language (ArSL).
Task
Build an end-to-end bidirectional sign language translation system capable of both recognizing signs from live video and generating them through an animated avatar.
Action
Trained a custom YOLOv8 model on a curated ArSL gesture dataset with real-time landmark extraction via MediaPipe. Achieved 91% gesture classification accuracy. Currently extending to 3D avatar rendering using skeletal animation pipelines, mapping predicted pose tensors to rigged character meshes for photorealistic sign output.
Result
Won 1st place at the P2M Innovation Competition. Ongoing R&D on avatar rendering represents a step change from 2D overlays to embodied, expressive sign generation — with patent potential.
YOLOv8 MediaPipe PyTorch 3D Avatar Rendering Skeletal Animation ArSL Dataset OpenCV
PROJECT_03
NASA Space Apps — Exoplanet Identification via ML
🚀 6th Global Astrophysics ML
Situation
NASA's Kepler/TESS datasets contain thousands of unvalidated planetary transit signals. Manual vetting by astronomers is a severe bottleneck — false positives (eclipsing binaries, instrumental artifacts) pollute candidate pools.
Task
Design a multi-source ML classifier that distinguishes genuine exoplanet transits from astrophysical false positives using heterogeneous time-series, spectral, and centroid data.
Action
Built a multi-modal ML pipeline fusing light curve time-series (1D-CNN), stellar metadata (gradient boosted features), and centroid shift analysis. Applied ensemble voting across models trained on NASA's cumulative KOI catalog. Implemented data augmentation via synthetic transit injection to address severe class imbalance.
Result
Achieved 89.4% validation accuracy on held-out KOI candidates. Ranked 6th globally at NASA Space Apps Challenge 2023, competing against 1,000+ teams across 160 countries.
1D-CNN Gradient Boosting Ensemble Methods NASA KOI Catalog TensorFlow Time-Series Analysis Python
PROJECT_04
DeepFake Voice Detection
🎙 Audio AI Deep Learning Binary Classifier
Situation
AI-generated voice cloning tools can now produce near-perfect imitations of real speakers — creating risks of misinformation, fraud, and identity deception at scale.
Task
Build a binary audio classifier that reliably distinguishes real human speech from AI-synthesized deepfake audio using acoustic feature extraction.
Action
Extracted 40 MFCC coefficients per audio file via Librosa. Handled class imbalance with RandomOverSampler. Trained a 4-layer Dense Neural Network (128 → 256 → 128 → softmax) with Dropout regularization and EarlyStopping on 300 epochs.
Result
Model correctly classified real vs. fake audio on unseen demonstration files — including celebrity voice clones (Linus → Musk). Validated on the Deep Voice Kaggle dataset across multiple speakers.
